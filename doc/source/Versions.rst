Code Versions
=============

The Global Chemical Equilibrium Code comes with different versions of the chemical network.
Currently the code contains:

 * The standard Version, see :ref:`network`
 * A carbon Version, see :ref:`Carbon_Version`
 * A Sulfur Nitrogen Version, see :ref:`Sulfur_Nitrogen_Version`
 * A Version similar to the Version used in :cite:p:`Young+2023`, see :ref:`Young_Version`
